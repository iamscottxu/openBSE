Class: exports

openBSE. exports

弹幕引擎对象类

new exports(element [, options] [, renderMode])

创建一个弹幕引擎对象。
Parameters:
Name Type Argument Default Description
element Element 要加载弹幕的元素：有关 Element 接口的信息请参阅MDN Element 。
options openBSE~generalOptions <optional>
 全局选项：一个 openBSE~generalOptions 结构。
renderMode string <optional>
 "canvas" 渲染模式：默认为“canvas”, “可选css3”， “webgl”和“svg”。一般建议使用“canvas”（特别是 FireFox 浏览器 CSS3 渲染效率较低）。在一些版本较老的浏览器中“window.devicePixelRatio”变量不被支持或支持不完整，这会导致在高DPI和页面被缩放的情况下“canvas”和“webgl”渲染模式弹幕显示不正常的情况（弹幕模糊），此时建议使用“css3”渲染模式。
Properties:
Name Type Description
options openBSE~Options 设置或获取全局选项。
visibility bool 获取或设置弹幕可见性。
renderMode string 获取渲染模式。取值为“canvas”、“css3”、“webgl”或“svg”。
playState bool 获取播放状态。true：正在播放；false：已暂停/停止播放。
debugInfo openBSE~DebugInfo 获取调试信息。

add(bulletScreen)

添加弹幕到弹幕列表。由于弹幕在屏幕上出现过后，弹幕引擎将从列表中彻底删除此弹幕。所以，在改变播放进度时，可能需要先清空弹幕列表，然后重新加载此播放进度以后的弹幕。

cleanBuffer()

清空弹幕列表，但屏幕上已经出现的弹幕不会被清除。

cleanScreen()

清空屏幕内容。清空屏幕上已经出现的弹幕，不包括弹幕列表中的弹幕。

pause()

暂停播放弹幕。暂停播放弹幕是指弹幕播放暂停，所有未出现的弹幕将停止出现，已出现的弹幕停止运动，停止消失。

play()

开始播放弹幕。

playAllBulletScreens()

继续所有在事件响应中设置了 e.pause = true; 弹幕的播放。

stop()

停止播放弹幕。停止播放弹幕是指停止播放弹幕，默认时间基准（options.clock）归零，并清空弹幕列表、清空屏幕内容。
