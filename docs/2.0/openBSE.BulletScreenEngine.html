Class: BulletScreenEngine

openBSE. BulletScreenEngine

弹幕引擎对象类

new BulletScreenEngine(element [, _options] [, renderMode])

创建一个弹幕引擎对象。
Parameters:
Name Type Argument Default Description
element Element 要加载弹幕的元素：有关 Element 接口的信息请参阅MDN Element 。
_options openBSE~Options <optional>
 全局选项：一个 openBSE~Options 结构。
renderMode string <optional>
 "canvas" 渲染模式：默认为“canvas”, “可选css3”， “webgl”和“svg”。一般建议使用“canvas”（特别是 FireFox 浏览器 CSS3 渲染效率较低）。在一些版本较老的浏览器中“window.devicePixelRatio”变量不被支持或支持不完整，这会导致在高DPI和页面被缩放的情况下“canvas”和“webgl”渲染模式弹幕显示不正常的情况（弹幕模糊），此时建议使用“css3”渲染模式。
Throws:
  • 浏览器不支持特定渲染模式时引发错误。
    Type
    openBSE.BrowserNotSupportError
  • 传入的参数错误时引发错误。请参阅 MDN TypeError 。
    Type
    TypeError

addBulletScreen(bulletScreen)

添加弹幕到弹幕列表。由于弹幕在屏幕上出现过后，弹幕引擎将从列表中彻底删除此弹幕。所以，在改变播放进度时，可能需要先清空弹幕列表，然后重新加载此播放进度以后的弹幕。

pause()

暂停播放弹幕。暂停播放弹幕是指弹幕播放暂停，所有未出现的弹幕将停止出现，已出现的弹幕停止运动，停止消失。

play()

开始播放弹幕。

playAllBulletScreens()

Events

click

弹幕单击事件。当单击弹幕时触发。
Parameters:
Name Type Argument Default Description
e.bulletScreen openBSE~BulletScreen <optional>
 被单击的弹幕的数据：一个 openBSE~BulletScreen 结构。可修改其中的参数，以便动态调整弹幕样式，但是一些参数在事件中修改无效，查看此结构的说明以了解详情。（注意：不要试图与添加弹幕时创建的对象进行比较，这个对象是克隆得到的，并不相等。正确的方法是在添加弹幕时一并插入 id 等自定义字段来唯一标识一条弹幕。）
e.redraw boolean <optional>
 false 是否重绘弹幕：此参数在每次引发事件时的初始值为 false ，如果修改了 e.bulletScreen 中的值，此参数必须设为 true ，否则修改无法生效。
e.pause boolean <optional>
 false 是否暂停：读取此参数可判断这条弹幕是否处于暂停状态，也可以设置此参数。如果设置为 true 则该弹幕暂停，直到下次引发任意弹幕事件时将此参数设为 false 或调用 openBSE.BulletScreenEngine#playAllBulletScreens 方法。
e.type string 事件类型（事件名称）
e.screenX number 当事件发生时，鼠标相对于显示器屏的 X 坐标。
e.screenY number 当事件发生时，鼠标相对于显示器屏的 Y 坐标。
e.clientX number 当事件发生时，鼠标相对于浏览器有效区域的 X 坐标。
e.pageX number 当事件发生时，鼠标相对于页面的 X 坐标。
e.pageY number 当事件发生时，鼠标相对于页面的 Y 坐标。
e.pageY number 当事件发生时，鼠标相对于页面的 Y 坐标。
Listeners of This Event:
  • openBSE.BulletScreenEngine#bind
