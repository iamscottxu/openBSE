Methods

addBulletScreen(bulletScreen)

添加弹幕到弹幕列表。由于弹幕在屏幕上出现过后，弹幕引擎将从列表中彻底删除此弹幕。所以，在改变播放进度时，可能需要先清空弹幕列表，然后重新加载此播放进度以后的弹幕。
Parameters:
Name Type Description
bulletScreen openBSE~BulletScreen 单条弹幕数据：一个 openBSE~BulletScreen 结构。

cleanBulletScreenList()

清空弹幕列表，但屏幕上已经出现的弹幕不会被清除。

cleanBulletScreenListOnScreen()

清空屏幕弹幕。清空屏幕上已经出现的弹幕，不包括弹幕列表中的弹幕。

pause()

暂停播放弹幕。暂停播放弹幕是指弹幕播放暂停，所有未出现的弹幕将停止出现，已出现的弹幕停止运动，停止消失。

play()

开始播放弹幕。

stop()

停止播放弹幕。停止播放弹幕是指停止播放弹幕，默认时间基准（options.clock）归零，并清空弹幕列表、清空屏幕弹幕。
